# dataverse-playground

## Install Dataverse extension from Google App Store 
The link is: https://chrome.google.com/webstore/detail/dataverse/kcigpjcafekokoclamfendmaapcljead

## Install dependencies

```bash
pnpm install
```

## Running the demo

```bash
pnpm dev
```

## Building the demo

```bash
pnpm build
```
